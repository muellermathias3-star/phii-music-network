# Music Distribution Outreach Pipeline

This workflow coordinates standard outreach for playlists, blogs, radio,
curators, and aligned discovery channels.

## Roles

- THOR researches targets, relevance, public context, and discovery signals.
- FREYA writes pitch text and promo copy in PHII's voice.
- LOKI coordinates outreach status, follow-ups, and deadlines.
- ODIN checks safety, plausibility, and strategic fit.

## Standard Outreach

Standard outreach can proceed when all conditions are true:

- No exclusivity
- No rights transfer
- No unusual terms
- No E2/E3 details
- No deadline pressure below the approval threshold
- FREYA tone check and ODIN plausibility check pass

## Non-Standard Outreach

Requires explicit approval before sending:

- Premieres or exclusive features
- Interviews with sensitive context
- Any rights, contract, fee, or long-term commitment question
- Requests involving unreleased restricted details
- Any unusual or high-risk ask

## Pipeline

1. THOR creates a target note with public context and relevance.
2. FREYA drafts a short, non-pushy pitch.
3. LOKI tracks send date, follow-up date, response, and next step.
4. ODIN reviews before external action.
5. Outcome is logged as a safe summary, not as private raw data.

## Tracking Fields

- Target name
- Public URL/contact route
- Fit reason
- Agent owner
- Status
- Last action date
- Next action date
- Safety notes
